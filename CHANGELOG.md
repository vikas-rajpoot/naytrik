## [1.0.0] - 2025-12-04

### Added
- **AI-Powered Recording**: Record browser workflows using Google's Gemini AI
- **Deterministic Playback**: Execute recorded workflows without AI API calls
- **Multi-Strategy Selector System**: CSS → XPath → Text → Coordinates fallback
- **Parameterized Workflows**: Support for variables and dynamic data injection
- **Screenshot Capture**: Visual debugging and documentation support
- **CLI Interface**: Command-line tools for recording, playback, and management
- **Python API**: Flexible programmatic integration
- **Playwright Integration**: Fast and reliable browser automation
- **Workflow Storage**: JSON/YAML persistence for recorded workflows

### Features
- `naytrik record` - Record new workflows with AI assistance
- `naytrik play` - Replay recorded workflows deterministically
- `naytrik list` - List all available workflows
- `naytrik show` - Display workflow details
